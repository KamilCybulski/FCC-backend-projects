XYZ image search abstraction layer

created by kamyl, see the source code

The main api endpoint is:

https://xyz/images

There are 3 query parameters:

  • q - mandatory, the search query
  • offset - optional, for pagination
  • count - optional, number of results

Example url:

https://xyz/images&q=cats&offset=10&count=20

Url to get 10 last search queries:

https://xyz/history
